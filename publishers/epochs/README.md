# Epoch Publisher

Publishes accepted or candidate epoch outputs toward ORBISTIUM when the runtime is authorized to do so.

Epoch publication writes state.
It does not define the law being written against.

## Deterministic epoch-candidate report

The runtime materializes a deterministic epoch-candidate report at:

reports/generated/sovereign-epoch-candidate.json

This report is derived from sovereign inventory and is a candidate output only.
It is not accepted state until ORBISTIUM records it.
